# kajkavski-rjecnik
Flutter aplikacija za rječnik kajkavskog narječja.

## Ideja
  - flutter aplikacija
  - flip kartice
  - lista rijeci
  - baza podataka iz lucijine excel tablice
  - online baza koja se pohranjuje na mobitel, aplikacija tijekom paljenja provjerava ima li razlika
  - reklame
